# DevConnect 🚀

A social networking platform built with Django.

## Features

* User Authentication (Login & Signup)
* Create, Edit and Delete Posts
* Like Posts
* Comment on Posts
* Upload Profile Pictures
* User Bio
* Follow / Unfollow Users
* Followers & Following Count
* Responsive Design for Mobile and Desktop

## Tech Stack

* Python
* Django
* HTML
* CSS
* SQLite
* Git & GitHub

## Installation

```bash
git clone https://github.com/HarshaPujar/DevConnect.git

cd DevConnect

python manage.py migrate

python manage.py runserver
```
